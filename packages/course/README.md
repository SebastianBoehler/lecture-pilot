# Course Package

This is a reserved package boundary, not a published or imported package.
Current course models, repository policy, source ingestion, lecture access, and
canvas publication are owned by `apps/api/src/lecturepilot/`; browser contracts
live in `apps/web/src/types.ts` and `apps/web/src/api.ts`.

Move code here only when a real cross-runtime generated contract exists. Do not
duplicate the current Python and TypeScript models merely to populate the
directory.
